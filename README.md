# Mini-Projet-Blog
Petit projet de dev back pour les cours.

**Url** : http://miniblog.robinvigier.fr/


## Fonctionnalité
### Conexion
- connexion
- création de compte
- déconnexion

### Billet
#### Pour le propriétaire :
- ajout d'un billet
- suppression d'un billet
- supression de tout les commentaires

#### Pour tout les utilisateurs :
- consulter un billet
- commenter un billet
- supprimer ses commentaires

### Autres
#### Pour le propriétaire :
- accéder à une vue administrateur
- peut y consulter l'ensemble des utilisateurs et les supprimer
- peut y consulter l'ensemble des billets et les supprimer
- peut y consulter l'ensemble des commentaires et les supprimer

#### Pour tout les utilisateurs :
- accéder à la page archives
